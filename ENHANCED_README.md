# ethereum-rwa-tokenization-template

Real-world asset (RWA) tokenization starter on Ethereum.

## Overview
Educational template for permissioned RWA tokenization with issuer controls and KYC-friendly design.

## Tech Stack
- Solidity
- Permissioned token patterns
- viem

## Getting Started
```bash
git clone https://github.com/Proxtox/ethereum-rwa-tokenization-template.git
cd ethereum-rwa-tokenization-template
npm install
```

## Key Files
- `contracts/RWATokenization.sol` - Basic minting with approved issuers
